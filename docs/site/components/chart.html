ComponentChart

Purpose

Chart.js-backed data visualization component rendered from Go props.

Key props

  • Type (line, bar, pie, doughnut, scatter)
  • Labels
  • Datasets (Label, Data or Points, colors, Fill, Tension)
  • Options (BeginAtZero, Stacked, HideLegend, axis labels)

Chart type examples

Use Type and dataset fields according to the chart purpose:

  • Line (ChartTypeLine): time-series trends (e.g., daily active users).
  • Bar (ChartTypeBar): category comparisons (e.g., revenue by plan).
  • Pie / Doughnut (ChartTypePie / ChartTypeDoughnut): composition ratios at a single point in time.
  • Scatter (ChartTypeScatter): correlation between two numeric dimensions.

Line chart (trend)

mf.ComponentChart(mf.ChartProps{
    Type:   mf.ChartTypeLine,
    Labels: []string{"Mon", "Tue", "Wed", "Thu", "Fri"},
    Datasets: []mf.ChartDataset{{
        Label: "Signups",
        Data:  []float64{12, 19, 14, 22, 18},
        Fill:  true,
    }},
})

Bar chart (comparison)

mf.ComponentChart(mf.ChartProps{
    Type:   mf.ChartTypeBar,
    Labels: []string{"Free", "Pro", "Enterprise"},
    Datasets: []mf.ChartDataset{{
        Label: "MRR",
        Data:  []float64{1200, 4800, 9200},
    }},
})

Pie / doughnut chart (composition)

mf.ComponentChart(mf.ChartProps{
    Type:   mf.ChartTypeDoughnut,
    Labels: []string{"Chrome", "Safari", "Firefox"},
    Datasets: []mf.ChartDataset{{
        Label: "Sessions",
        Data:  []float64{62, 24, 14},

})

Scatter chart (correlation)

mf.ComponentChart(mf.ChartProps{
    Type: mf.ChartTypeScatter,
    Datasets: []mf.ChartDataset{{
        Label: "Campaign A",
        Points: []mf.ChartPoint{
            {X: 1000, Y: 45},
            {X: 2000, Y: 73},
            {X: 3000, Y: 96},
        },
    }},
})

Visual

Go usage

signupsChart := mf.ComponentChart(mf.ChartProps{
    Type:        mf.ChartTypeLine,
    Title:       "Weekly signups",
    Description: "New accounts by weekday.",
    Labels:      []string{"Mon", "Tue", "Wed", "Thu", "Fri"},
    Datasets: []mf.ChartDataset{
        {
            Label:           "Signups",
            Data:            []float64{12, 19, 14, 22, 18},
            BorderColor:     "#2563eb",
            BackgroundColor: "rgba(37, 99, 235, 0.16)",
            Fill:            true,
            Tension:         0.3,
        },
    },
    Options: mf.ChartOptions{
        BeginAtZero: true,
        YAxisLabel:  "Users",
    },
})
