ComponentRadioGroup

Purpose

Exclusive selection group rendered from item definitions.

Key props

  • Name, AriaLabel
  • Items (Label, Value, Checked, Disabled)
  • Props (Size, Disabled, Class)

Visual

Go usage

roleGroup := mf.ComponentRadioGroup(mf.RadioGroupComponentProps{
    Name:      "role",
    AriaLabel: "role",
    Items: []mf.RadioItem{
        {Label: "Admin", Value: "admin", Checked: true},
        {Label: "Editor", Value: "editor"},
        {Label: "Viewer", Value: "viewer", Disabled: true},
    },
    Props: mf.ComponentProps{Size: "sm"},
})
