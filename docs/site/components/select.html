ComponentSelect

Purpose

Dropdown input for selecting one option from a list.

Key props

  • name
  • options (label, value, selected)
  • Props (Variant, Size)

Visual

Go usage

roleSelect := mrn.ComponentSelect("role", []mrn.SelectOption{
    {Label: "Admin", Value: "admin", Selected: true},
    {Label: "Viewer", Value: "viewer"},
}, mrn.ComponentProps{
    Variant: "ghost",
    Size:    "sm",
})
