ComponentStack

Purpose

Flex stack for vertical and horizontal layout rhythm.

Key props

  • Direction (vertical, horizontal)
  • Gap (none, xs, sm, md, lg, xl)
  • Align, Justify, Wrap
  • Props.Class

Visual

Go usage

stack := mrn.ComponentStack(
    mrn.StackProps{
        Direction: "horizontal",
        Gap:       "sm",
        Align:     "center",
        Justify:   "between",
        Wrap:      true,
    },
    mrn.Text("Aiko Tanaka"),
    mrn.DivClass("badge badge-primary", mrn.Text("Admin")),
    mrn.ComponentButton("Open", mrn.ComponentProps{Variant: "secondary", Size: "sm"}),
)
