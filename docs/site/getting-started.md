# Quickstart

This guide gets you from clone to running demo quickly.

## 1. Run the admin demo

```bash
go run ./cmd/marionette
```

Open `http://127.0.0.1:8080`.

## 2. Run the minimal sample

```bash
go run ./cmd/simple-sample
```

Open `http://127.0.0.1:8081`.

## 3. Optional: hot reload for development

```bash
go install github.com/air-verse/air@latest
air
```
